## Structure
```python
import os

class MainClass:

    CONSTANTS = 'Constant Value'

    def __init__(self):
        self.attribute = None

    @staticmethod
    def this_is_static_method():
        return "static method return"

    def this_is_instance_method(self):
        return "instance method return"
```
